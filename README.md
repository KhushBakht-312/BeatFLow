BeatFlow - AI-powered ECG Analyzer
BeatFlow is a web application that allows users to upload ECG images and get instant insights about heart health using AI. The project is trained on many dataset.

## Features
 Detects and Classify ECG as Healthy or Unhealthy.
 History section for last 3 uploads.
 Downloadable report for analysis results.
 User-friendly 

Libraries and Languages used:
 Python, Flask
 PyTorch, Torchvision
 HTML, CSS, JavaScript

## How to Run
 Clone the repository.
 Create a virtual environment in `backend` and install dependencies:
   pip install -r backend/requirements.txt
